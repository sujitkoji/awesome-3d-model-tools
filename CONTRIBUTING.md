# Contributing

Thank you for your interest in contributing to **Awesome 3D Model Tools**!

We welcome high-quality contributions related to **GLB**, **glTF**, **3D graphics**, **WebGL**, **WebGPU**, **Three.js**, **Babylon.js**, **React Three Fiber (R3F)**, **viewers**, **validators**, **converters**, **optimizers**, **libraries**, **SDKs**, and other developer tools.

## Accepted Categories

Please add tools only if they fit one of the following categories:

- Browser Viewers
- Desktop Viewers
- Screenshot Tools
- Validators & Analyzers
- Optimizers & Compression
- Converters & Editors
- Libraries & SDKs
- Official Resources
- Learning Resources


## Before Submitting

Please make sure the tool:

- Is actively maintained.
- Has an official website or GitHub repository.
- Fits one of the repository categories.
- Includes a short, clear, and objective description.
- Is not already listed.
- Provides real value to developers or 3D artists.

## Tool Format

Please use the following format when adding a new tool.

```md
### Tool Name

Short one-line description.

- Website:
- GitHub:
- License:
- Platform:
- Pricing:
- Best For:
```

Example:

```md
### Example Viewer

Browser-based GLB and glTF viewer for inspecting 3D assets.

- Website: https://example.com
- GitHub: https://github.com/example/viewer
- License: MIT
- Platform: Web
- Pricing: Free
- Best For: Viewing, Debugging, Inspection
```

## Pull Requests

Before opening a Pull Request:

- Keep descriptions concise and factual.
- Prefer official documentation over third-party links.
- Verify that all links are working.
- Follow the existing formatting and style.
- Avoid duplicate entries.
- Group similar additions together when possible.

## Repository Guidelines

Please:

- Keep descriptions neutral and informative.
- Avoid promotional language.
- Do not add referral or affiliate links.
- Prefer official websites and official GitHub repositories.
- Keep entries organized alphabetically whenever possible.

## Reporting Issues

If you find:

- Broken links
- Outdated information
- Incorrect descriptions
- Duplicate tools

Please open an Issue or submit a Pull Request.


## Thank You

Every contribution helps make **Awesome 3D Model Tools** a better resource for the global 3D development community.

Thank you for contributing! 🚀
