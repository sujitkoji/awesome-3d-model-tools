# Contributing

Thank you for your interest in contributing!

We welcome high-quality additions related to 3D graphics, GLB, glTF, WebGL, Three.js, Babylon.js, React Three Fiber, and related tools.

## Before submitting

Please make sure the tool:

- Is actively maintained.
- Has an official website or GitHub repository.
- Fits one of the repository categories.
- Includes a short, clear description.
- Avoids duplicate entries.

## Format

## Pull Requests

Please:

- Keep descriptions concise.
- Prefer official documentation.
- Verify all links work.
- Follow existing formatting.

Thank you for helping improve this repository.
